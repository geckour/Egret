# Egret
Mastodonと遊ぶどん！

## What's this?
MastodonのAndroidクライアントです。

## Features
- Public timelineのストリーム表示
- User timelineのストリーム表示
- 複数アカウントの切り替え
- アカウントプロフィールの表示
- アカウントの(アン)フォロー・ミュート・ブロック
- トゥートの投稿(文字のみ)
- トゥートへの返信の投稿(文字のみ)
- トゥートのお気に入り・ブースト
- and more in future...
